# Requisition App

Cross-platform Expo app for material and equipment requisitions.

## Run

```bash
npm install
npm start
```

Use the Expo Go app to open it on Android or iOS, or run:

```bash
npm run android
npm run ios
```

## Roles

- Manager: add, edit, remove, and update status.
- Procurement: update status only.
- Engineer: add, edit, and remove.
- Staff: add and edit.

All roles can view requisition details and statuses.
